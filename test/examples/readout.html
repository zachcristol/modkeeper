Project Description

  • Goal: provide easier path to bring internal applications to the cloud
  • Creating repository of modules for configurable cloud infrastructures
  • Infrastructure as a service (iaas): Provide a simple abstraction to create a suite of infrastructure

Use Case

  • There is a team that is creating an app or has an app on prem
  • The team sees benifits of cloud applications (flexible cost, highly configurable, high reliability, high mobility)
  • When the team needs to set up a suite of infrastructure (Cloud Endpoints, Appengine, Cloud SQL, Storage Bucket), they can dictate the configuration of the infrastructure in the form of a module.
  • These modules (configurations) can then be stored on our application and be used and tweaked for others to use

Buisness Values

  • Deploy to cloud without the headache
  • Have developers spend less time on the command line and more time in code
  • Simple, repeatable, configurable: Creating a rich community of module creators and users for internal developers
  • We ARE a use case (Cloud Endpoints, Appengine, Cloud SQL, Storage Bucket)
